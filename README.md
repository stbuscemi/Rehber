# Rehberim 🌟

Günlük durumlarını dini, felsefi ve ezoterik kaynaklarla yorumlayan uygulama.

## Yayınlama Rehberi (Adım Adım)

### 1. GitHub Hesabı Aç
- https://github.com adresine git
- "Sign up" ile hesap oluştur

### 2. Yeni Repo Oluştur
- Sağ üstteki "+" → "New repository"
- Repository name: `rehberim`
- "Create repository" butonuna bas

### 3. Dosyaları Yükle
- "uploading an existing file" linkine tıkla
- Bu klasördeki tüm dosyaları sürükle-bırak
- "Commit changes" butonuna bas

### 4. Vercel'e Deploy Et
- https://vercel.com adresine git
- "Continue with GitHub" ile giriş yap
- "New Project" → `rehberim` reposunu seç
- "Import" butonuna bas

### 5. API Key Ekle (ÖNEMLİ)
- Deploy ekranında "Environment Variables" bölümüne bak
- Name: `ANTHROPIC_API_KEY`
- Value: Anthropic API key'ini buraya yapıştır
- "Deploy" butonuna bas

### 6. Uygulaman Yayında! 🎉
- `rehberim.vercel.app` gibi bir link alacaksın
